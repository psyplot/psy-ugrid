Installation

Warning

This page has been automatically generated as has not yet been reviewed by the authors of psy-ugrid!

To install the psy-ugrid package, we recommend that you install it from PyPi via:

pip install psy-ugrid

Or install it directly from the source code repository on Gitlab via:

pip install git+https://codebase.helmholtz.cloud/psyplot/psy-ugrid.git

The latter should however only be done if you want to access the development versions.
